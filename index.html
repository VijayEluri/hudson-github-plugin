github plugin for the Hudson continous integration server.

The plugin adds a sidebar link to a github enabled project. Changes will link back to the github commit page.

The current release is 0.1 - For more information have a look at the github plugin page.

The plugin is available from the official Hudson plugin repository. Go to the Hudson Plugin-Manager and install the Github plugin. Don't forget to restart Hudson!

Download

You can download the plugin file (.hpi) from the Hudson download page.

You can also clone the project with Git by running: 

$ git clone git://github.com/juretta/hudson-github-plugin
